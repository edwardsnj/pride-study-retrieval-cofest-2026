# Pride Study Document Retrieval Problem - BOSC CoFest2026
Repository for the PRIDE Study Document Retrieval Problem team at BOSC CoFest 2026.

Email nje5@georgetown.edu to be added to the project.

## Objective
Identify [PRIDE studies](https://www.ebi.ac.uk/pride/) that use specific sample-preparation, instrumentation, or data-processing steps not described by the PRIDE data-model from the text-based study summaries, for tagging or reanalysis.

## Specific Use Case
De-glycosylated N-glycopeptides, based on the use of PNGase F and variable modification deamidation in data analysis. 

## Dataset
36K PRIDE JSON documents, plus openai-3-small text embeddings. A small number of curated true positives (~80) are available. See [data](data).
